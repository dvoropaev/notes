Причины возникновения прерывания:
- Внешнее устройство требует вгимания (асинхронное прерывание)
- Произошла ошибка при выполнении инструкции (синхронное прерывание)
- Специальная инструкция (програмные)

Асинхроные прерывания:
- код не готов к тому. что его прервут
- обработчик прерывания отвечает за сохранение состояния первой задачи

Маскируемые прерывания - можно запрещать
Немаскируемые прерывания - обрабатываются всегда
cli - запретить прерывания (0 -> IF)
sti - разрешить прерывания (1 -> IF)

Вектор прерывания — номер вектора прерываний для заданного устройства. Таблица (IDT) векторов прерываний содержит адреса обработчиков.

Адрес IDT хранится в IDTR 
LIDT и SIDT чтение и запись в IDTR
в IDT может быть только 256 записей 

стек:
SS
RSP
RFLAGS
CS
RIP
(не обязательно)Error Code

за сохраннение состояния остальных регистро отвечает обработчик

iretq - инструкция завершения выполнения прерывания
